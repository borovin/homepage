Разработка мультиоконного онлайн-терминала для валютной биржи (Single Page Application, WebSockets).
Работа с биржевым сервером через WebSockets (SocketIO).
Шаблонизация и рендеринг высокодинамичных данных на стороне клиента (KnockoutJS, CanJS).
Отображение этих данных в виде сложных графиков (Highcharts).
Проектирование модульной мультиоконной архитектуры (BEM, RequireJS).
